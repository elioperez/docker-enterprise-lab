# Docker Enterprise Lab

> A hands-on Docker portfolio built by following the **DevOps with Docker** course from the **University of Helsinki (MOOC.fi)**.

---

## About This Repository

This repository documents my journey of learning Docker through practical, enterprise-oriented projects.

Rather than simply following tutorials, every project is designed to simulate real-world Linux administration tasks that a Junior Linux Administrator or Junior DevOps Engineer may perform in a production environment.

Each lab follows a structured mentoring methodology that combines theory, hands-on practice, troubleshooting, technical documentation, GitHub workflow, and interview preparation.

---

## Learning Methodology

Every project follows the same workflow.

```
MOOC.fi ( University of Helsinki – DevOps with Docker )
      ↓
Mentor Questions
      ↓
Linux Lab
      ↓
Troubleshooting
      ↓
README
      ↓
GitHub (Windows Host)
      ↓
Interview English
```

This methodology allows me not only to learn Docker, but also to develop the mindset and documentation practices expected from a professional Linux Administrator.
