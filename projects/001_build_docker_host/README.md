# Project 001 – Preparing a Linux Server for Docker

## Overview

This project is the first lab of my **Docker Enterprise Lab Portfolio**, based on the **DevOps with Docker** 

The objective of this project was **not** to install Docker, but to prepare a dedicated Linux server following enterprise Linux administration best practices before deploying Docker Engine.

This project simulates a real-world scenario where a Linux Administrator must validate the operating system before introducing new infrastructure software into a production environment.

---

## Project Goals

- Verify the operating system
- Verify the Linux kernel version
- Configure the server hostname
- Verify network configuration
- Test Internet connectivity
- Update the operating system
- Install essential Linux administration tools
- Verify the SSH service
- Prepare the server for Docker Engine installation
